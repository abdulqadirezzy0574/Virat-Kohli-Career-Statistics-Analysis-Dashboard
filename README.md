# Virat-Kohli-Career-Statistics-Analysis-Dashboard

## Project Objective
The objective of this project is to analyze Virat Kohli's international cricket career statistics using SQL and Power BI to gain meaningful insights into his performance across different formats. The project focuses on evaluating batting performance, yearly run trends, batting average, strike rate, centuries, half-centuries, and overall career achievements through interactive dashboards and data analysis.

By transforming raw cricket data into meaningful visualizations, this project demonstrates the use of SQL for data analysis and Power BI for dashboard creation, helping uncover performance patterns and career milestones through data-driven insights.
